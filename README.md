# TravelThemes

TravelThemes is a photo-centric travel website where trips are displayed through themed visual presentations. Each trip can have a different theme that controls how trips are shown.

## Tech Stack

| Technology | Version |
|------------|---------|
| Next.js | 16.1.3 |
| React | 19.2.3 |
| TypeScript | 5.x |
| Tailwind CSS | 4.x |
